Help

Cosym coordinates

This graph shows the result of the coordinate method of clustering.

The further away from the origin a dataset is, the less random error it has. The smaller the angle between pairs of datasets, the less systematic error there is between them. This means that datasets that are internally consistent will cluster together.

The data are automatically classified into clusters based on their spatial density in this plot. Classification is performed using the OPTICS method (see the OPTICS reachability plot).

Note that higher dimensions may be present, but this graph only shows the first two principal components.

This coordinate based approach is described in more detail here.

Highlight a subsection using the box or lasso selection tools to produce a xia2.multiplex command to merge your selection.

OPTICS reachability

This graph shows the density structure of the cosym coordinates.

OPTICS is a density-based clustering method that orders datasets based on their density structure. This produces a reachability plot, where valleys correspond to high desity regions. Peaks between clusters indicate boundaries or outliers. The start of a cluster begins with a high reachability distance, because this indicates it is far from the preceeding dataset, not the next dataset.

Highlight a subsection using the box or lasso selection tools to produce a xia2.multiplex command to merge your selection.

Correlation coefficient clustering

This graph shows the result of hierarchical clustering based on pairwise correlation coefficients between datasets. The heatmap shows the correlation coefficients, while the dendrograms show the hierarchical clustering structure. Datasets which connect higher up in the dendrogram are more different to each other.

Click on a branch in the dendrogram to produce a xia2.multiplex command to merge that cluster.

Cos angle clustering

This graph shows the result of hierarchical clustering based on pairwise cosine angles between datasets in the cosym coordinate plot. The heatmap shows the cosine angles, while the dendrograms show the hierarchical clustering structure. Datasets which connect higher up in the dendrogram are more systematically different to each other.

Click on a branch in the dendrogram to produce a xia2.multiplex command to merge that cluster.

Principal component analysis

This matrix-style graph is an extension of the cosym coordinate plot to show between 2 and 6 dimensions. If more than 6 dimensions are present, only the first 6 are shown.

Note that selecting datasets using the box or lasso tool on one graph will highlight the datasets across all dimensions.

Highlight a subsection using the box or lasso selection tools to produce a xia2.multiplex command to merge your selection.
